# **Variable (변수)**

- 필요한 값을 일시적으로 저장하기 위해 사용.
- 데이터형을 개발자가 지정할 수 있다.
    - 데이터형의 결정은 값이 할당될 때
- 전역변수, 지역변수 2가지로 구분.
    - 전역변수 : 어디에서든 사용 가능
        - 변수명 = 값;
        - 문법

```jsx
1. 선언+값할당 : 변수명 = 값;
2. 값할당 : 변수명 = 값;
3. 값사용 : 출력, 연산, 재할당
```

- 지역변수 : 함수안에서 사용
    - var 변수명;
    - 문법

```jsx
1. 선언 : var 변수명;
2. 값할당 : 변수명 = 값;
3. 값사용 : 출력, 연산, 재할당
```
